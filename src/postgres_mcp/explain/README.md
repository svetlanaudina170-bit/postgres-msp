Описание файла:
Файл README.md предоставляет краткую документацию для модуля PostgreSQL Explain Tools, описывая его назначение,
основные инструменты и способы использования. Модуль предназначен для анализа планов выполнения запросов
в PostgreSQL, предоставляя функциональность для генерации и анализа EXPLAIN-планов, включая возможность
тестирования гипотетических индексов. Документ структурирован с использованием Markdown и включает разделы
о инструментах, использовании и преимуществах.

- Описание модуля и его назначения

- Преимущества использования модуля

Модуль интегрирован с PostgreSQL MCP server и предполагает наличие доступа к API MCP.

PostgreSQL Explain Tools
Этот модуль предоставляет инструменты для анализа планов выполнения запросов PostgreSQL.
Инструменты
ExplainPlanTool
Предоставляет методы для генерации различных типов планов EXPLAIN.
Использование
Инструмент explain интегрирован в сервер PostgreSQL MCP и может быть использован через API MCP с помощью функции:

explain_query

Эта функция принимает параметры для управления поведением:

sql — SQL-запрос для анализа (обязательный)
analyze — Если true, выполняет запрос для получения реальной статистики (по умолчанию: false)
hypothetical_indexes — Необязательный список индексов для симуляции без их создания

Преимущества

Понимание запросов: Помогает понять, как PostgreSQL выполняет запросы
Анализ производительности: Выявляет узкие места и возможности для оптимизации
Тестирование индексов: Позволяет тестировать гипотетические индексы без их фактического создания
